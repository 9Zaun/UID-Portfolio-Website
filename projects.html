Reinforcement Learning

Autonomous Tactical Agent

Multi-agent RL system for real-time strategic decision-making in adversarial environments.

Built a multi-agent reinforcement learning framework that trains autonomous agents to make tactical decisions in competitive environments with limited information. The system uses a custom reward architecture, combining objective completion with survivability heuristics. Agents learn emergent cooperative strategies (flanking, resource denial, feints) without explicit programming. Trained on a custom simulation engine with fog-of-war mechanics, and the system achieves a 74% win rate against scripted baselines after 2M episodes.

Defense Tech

Sentinel

Low-latency threat classification pipeline for edge-deployed sensor networks.

Designed and prototyped a real-time threat classification system built for edge deployment on resource-constrained hardware. The pipeline ingests multi-modal sensor data (such as infrared, acoustic, and RF) and runs inference through a quantized transformer model optimized for sub-10ms latency. Includes a custom alert prioritization layer that reduces false-positive rates by 38% compared to threshold-based baselines. Built with C++ and ONNX Runtime, targeting ARM-based field units.

Quantum Communication

Photon

QKD protocol simulator for noise-resilient quantum key distribution over fiber channels.

Developed a simulation framework for BB84 and E91 quantum key distribution protocols with realistic noise models- photon loss, detector dark counts, and polarization drift. The simulator benchmarks key generation rates and quantum bit error rates across variable fiber distances (up to 200km). Includes an adaptive basis reconciliation module that improves secure key yield by 22% under high-noise conditions. Written in Python with NumPy-accelerated state evolution and a custom channel model.

Systems & Tooling

Forge

Compiled DSL for defining and deploying inference graphs on heterogeneous hardware.

Built a domain-specific language and compiler toolchain for defining machine learning inference pipelines that target heterogeneous hardware- CPU, GPU, and custom accelerators. Forge programs compile to an intermediate representation, undergo graph-level optimizations (operator fusion, memory planning, scheduling), and emit platform-specific kernels. The runtime handles dynamic batching and zero-copy tensor passing between execution units. Early benchmarks show 2.3× throughput improvement over hand-written CUDA pipelines for common vision models.
